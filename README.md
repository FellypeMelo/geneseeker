# GeneSeeker

Identificador de Open Reading Frames (ORFs) nos 3 quadros de leitura.

## Instalação

```bash
pip install -r requirements.txt
```

## Uso

Execute: `python main.py`

## Funcionalidades

- Identifica códons START (ATG)
- Identifica códons STOP (TAA, TAG, TGA)
- Analisa todos os 3 quadros de leitura
- Gera relatório estruturado
